Start from zero. Learn bash scripting step-by-step with hands-on examples. Perfect for complete beginners — no prior experience needed.

6 lessons • 30 minutes • 100% beginner-friendly

Lesson 1: Create Your First Script

Let's create your very first bash script! A bash script is just a text file with commands that the computer will run one after another.

Step 1: Create a new file
Open your terminal and type:
nano hello.sh
This creates a new file called hello.sh (the .sh means "shell script")
Step 2: Type this code
Copy and paste this exact code into the editor:
#!/bin/bash echo "Hello, World!"
What does each line mean?
• #!/bin/bash — This tells your computer to use bash to run this script. Always put this on line 1!
• echo — This prints text to the screen
• The text in quotes is what gets printed
Step 3: Save and exit
Press Ctrl+X, then Y, then Enter to save the file
Step 4: Run your script
Type this in the terminal:
bash hello.sh
Output:
Hello, World!
⚠️ Common Mistake:
Don't forget the #!/bin/bash line at the very beginning! This tells the computer which program to use to run your script.
Try It Yourself
Create a script that prints your name and favorite food. Call it about_me.sh
#!/bin/bash echo "My name is Alice" echo "My favorite food is pizza"

Lesson 2: Variables (Storing Information)

Variables are like boxes where you store information. You can use them later in your script.

Create a new file called variables.sh
#!/bin/bash name="Alice" age=25 city="New York" echo "My name is $name" echo "I am $age years old" echo "I live in $city"
Save this and run it with: bash variables.sh
Output:
My name is Alice
I am 25 years old
I live in New York
How variables work:
• name="Alice" — Creates a variable named "name" and puts "Alice" in it
• $name — Gets the value from the variable (the dollar sign is VERY important!)
• Variables can hold text (strings) or numbers
• No spaces around the equals sign!
🔴 This will NOT work:
name = "Alice" ← spaces around = will cause an error
echo name ← missing $ will just print the word "name"
Variable naming rules
• Use lowercase letters: my_name ✓
• Can use numbers: name1, name2 ✓
• Cannot start with a number: 1name ✗
• Cannot use spaces: my name ✗
• Use underscores for spaces: my_name ✓
Try It Yourself
Create a script that stores your favorite color and hobby, then prints them out in a sentence.
#!/bin/bash color="blue" hobby="painting" echo "My favorite color is $color" echo "My hobby is $hobby"

Lesson 3: Getting Input From Users

So far our scripts are boring because they always do the same thing. Let's make them ask the user for information!

Create a file called greet.sh
#!/bin/bash echo "What is your name?" read name echo "Nice to meet you, $name!" echo "Welcome to bash scripting!"
Save and run it: bash greet.sh
Output (what you see):
What is your name?
_ ← cursor waits for you to type
How read works:
• read name — Waits for the user to type something
• Stores whatever they type in a variable called "name"
• The user presses Enter when they're done typing
• Then the script continues
Better way: Use a prompt
Instead of using echo and then read on separate lines, do this:
#!/bin/bash read -p "What is your name? " name echo "Nice to meet you, $name!"
The -p flag lets you show a message while waiting for input. Much cleaner!
💡 Pro Tip:
Always put your variable in quotes when using it: echo "Hello, $name"
This prevents problems if the input contains special characters.
Try It Yourself
Create a script that asks the user for their favorite food, then says something like "I love pizza too!"
#!/bin/bash read -p "What is your favorite food? " food echo "I love $food too!"

Lesson 4: Making Decisions (if/then)

Now let's make scripts that can make decisions. "If the user is an adult, print one thing. Otherwise, print something different."

Create a file called age_check.sh
#!/bin/bash read -p "How old are you? " age if [ $age -ge 18 ]; then echo "You are an adult!" else echo "You are a minor." fi
Run it: bash age_check.sh
If you enter 20:
You are an adult!
If you enter 12:
You are a minor.
The if/then/else structure:
• if [ condition ]; — Check if something is true
• then — What to do if it's true
• else — What to do if it's false
• fi — End the if block (fi = "if" backwards!)
Comparison operators
For numbers:
• -eq — equal to
• -ne — NOT equal to
• -lt — less than
• -le — less than or equal
• -gt — greater than
• -ge — greater than or equal

For text:
• = — equal to (note: just one = for text)
• != — NOT equal to
⚠️ Important:
Look carefully at the spaces: [ $age -ge 18 ]
There MUST be spaces after [ and before ]. This confuses beginners a lot!
Try It Yourself
Create a script that asks for a password. If they enter "secret123", print "Access granted!". Otherwise, print "Access denied!"
#!/bin/bash read -p "Enter password: " password if [ "$password" = "secret123" ]; then echo "Access granted!" else echo "Access denied!" fi

Lesson 5: Loops (Repeat Code)

Loops let you repeat code multiple times. Perfect for doing something again and again without typing the same thing over and over.

Create a file called countdown.sh
#!/bin/bash for i in 5 4 3 2 1 do echo $i done echo "Blastoff!"
Run it: bash countdown.sh
Output:
5
4
3
2
1
Blastoff!
How the for loop works:
• for i in 5 4 3 2 1 — i takes each value: 5, then 4, then 3, etc.
• do — Start the loop
• echo $i — Print the current value
• done — End the loop
Better way: Use ranges
Instead of listing numbers, use a range:
#!/bin/bash for i in {1..5} do echo "Number: $i" done
The {1..5} means "all numbers from 1 to 5"
Output:
Number: 1
Number: 2
Number: 3
Number: 4
Number: 5
Loop over a list
You can loop over words too:
#!/bin/bash for fruit in apple banana orange do echo "I like $fruit" done
Output:
I like apple
I like banana
I like orange
Try It Yourself
Create a script that prints the times table for 3 (3×1=3, 3×2=6, ... 3×10=30)
#!/bin/bash for i in {1..10} do result=$((3 * i)) echo "3 x $i = $result" done

Lesson 6: Your First Real Script

Let's put everything together and create a practical script that does something useful!

Create a file called guess_game.sh
This is a simple guessing game:
#!/bin/bash echo "Welcome to the number guessing game!" echo "" secret=42 guesses=0 while true do read -p "Guess a number between 1 and 100: " guess guesses=$((guesses + 1)) if [ $guess -eq $secret ]; then echo "You got it! The number was $secret" echo "It took you $guesses tries" break elif [ $guess -lt $secret ]; then echo "Too low! Try again." else echo "Too high! Try again." fi done
Run it: bash guess_game.sh
New things in this script:
• while true — Loop forever (until we use break)
• guesses=$((guesses + 1)) — Add 1 to the counter
• break — Exit the loop immediately
• elif — "Else if" for more conditions
Let's break down what happens:
1. Script asks the user to guess a number
2. Count how many tries they've had
3. If they guessed right, congratulate them and show the tries
4. If their guess is too low, tell them
5. If their guess is too high, tell them
6. Go back to step 1 and ask again
7. When they guess correctly, use break to stop the loop
Challenge: Improve the Game
Modify the script to:
1. Let the player choose the difficulty (easy = 1-20, medium = 1-50, hard = 1-100)
2. Give them a maximum of 5 tries
3. If they run out of tries, tell them the answer
#!/bin/bash echo "Welcome to the guessing game!" echo "" read -p "Choose difficulty (easy/medium/hard): " difficulty if [ "$difficulty" = "easy" ]; then max_num=20 max_tries=5 elif [ "$difficulty" = "medium" ]; then max_num=50 max_tries=5 else max_num=100 max_tries=5 fi secret=$((RANDOM % max_num + 1)) guesses=0 while [ $guesses -lt $max_tries ] do read -p "Guess (1-$max_num): " guess guesses=$((guesses + 1)) if [ $guess -eq $secret ]; then echo "Correct! It took $guesses tries!" exit 0 elif [ $guess -lt $secret ]; then echo "Too low!" else echo "Too high!" fi done echo "Out of tries! The answer was $secret"
💡 New concept: RANDOM
RANDOM is a bash variable that gives a random number
$((RANDOM % 100)) gives a number from 0-99
This is a built-in feature of bash!
