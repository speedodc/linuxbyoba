section 01

What is Linux?

Linux is an open-source operating system kernel first created by Linus Torvalds in 1991. Today it powers everything from Android phones to web servers, supercomputers, and cloud infrastructure.

When people say "install Linux" they usually mean installing a Linux distribution (distro) — the kernel bundled with a desktop environment, package manager, and default apps.

💡
Linux is free. You don't need a product key, activation, or a subscription. Download, install, and use it forever.

section 03

Install Ubuntu (Full Install)

Step-by-step guide to installing Ubuntu alongside or instead of Windows.

01

Download the Ubuntu ISO

Go to ubuntu.com/download/desktop and download the latest LTS version. The file will be around 5 GB.

📁
An ISO file is a complete image of a disc. You'll flash it onto a USB drive to create a bootable installer.
02

Flash the ISO to a USB Drive

You need a USB drive with at least 8 GB of space. Flashing will erase everything on it.

Download balenaEtcher (free, works on Windows/Mac/Linux):

  1. Open balenaEtcher
  2. Click "Flash from file" → select your Ubuntu .iso
  3. Click "Select target" → choose your USB drive
  4. Click "Flash!" and wait ~5 minutes
03

Boot from the USB Drive

Restart your computer and enter the BIOS/UEFI boot menu. The key to press depends on your manufacturer:

BrandKey to press at startup
DellF12
HPF9 or Esc
LenovoF12 or Fn+F12
ASUSF8 or Esc
AcerF12
MSIF11

Select your USB drive from the boot menu. Ubuntu will load a live environment.

04

Run the Ubuntu Installer

  1. Click "Install Ubuntu" on the desktop
  2. Select your language and keyboard layout
  3. Choose "Normal installation" and check "Download updates while installing"
  4. For installation type, choose:
    • "Install alongside Windows" — keeps Windows, dual-boot (recommended if unsure)
    • "Erase disk and install Ubuntu" — replaces Windows completely
  5. Set your timezone, name, username, and password
  6. Click Install and wait ~15–20 minutes
  7. Restart when prompted. Remove the USB drive when asked.
⚠️
Back up your data first! If you choose "Erase disk", all data on that drive will be permanently deleted.
05

First Boot — Update Everything

After logging into Ubuntu for the first time, open the terminal (Ctrl + Alt + T) and run:

bash
sudo apt update && sudo apt upgrade -y

This updates the package list and installs all available upgrades. Always do this after a fresh install.

section 04

Ubuntu on Windows (WSL via PowerShell)

Run a full Ubuntu Linux terminal inside Windows 10/11 without dual-booting or a VM. This is the fastest way to get started.

🪟
WSL (Windows Subsystem for Linux) is a feature built into Windows 10/11 that lets you run a real Linux environment directly inside Windows. No reboot needed — it opens like any other app.
01

Open PowerShell as Administrator

  1. Press Win key and search PowerShell
  2. Right-click Windows PowerShell
  3. Click "Run as administrator"
  4. Click Yes on the UAC prompt
⚠️
You must run as Administrator or the install command will fail with a permissions error.
02

Install WSL + Ubuntu with One Command

In the Administrator PowerShell window, paste this single command and press Enter:

powershell
wsl --install

This automatically:

  • Enables the WSL feature
  • Enables the Virtual Machine Platform
  • Downloads and installs Ubuntu (latest LTS)
ℹ️
The download is around 500 MB. It may take a few minutes depending on your connection speed.
03

Restart Your Computer

After the install finishes, you'll see this message:

output
The requested operation is successful. Changes will not be effective until the system is rebooted.

Save any open work and restart your computer. WSL won't work until you do.

04

Set Up Your Ubuntu User

After restarting, Ubuntu will open automatically and ask you to create a user account:

ubuntu setup
Enter new UNIX username: yourname
New password: ********
Retype new password: ********
passwd: password updated successfully
  • Choose any lowercase username (no spaces)
  • The password won't show as you type — that's normal
  • This password is used for sudo commands
05

Install a Specific Ubuntu Version (Optional)

Want Ubuntu 22.04 instead of the default? You can choose:

powershell
# List all available Linux distros
wsl --list --online

# Install a specific version
wsl --install -d Ubuntu-22.04
06

Useful WSL Commands in PowerShell

You can manage WSL at any time from PowerShell (no admin needed for these):

CommandWhat it does
wslOpen the default Linux distro
wsl --list --verboseShow installed distros and their status
wsl --shutdownStop all running WSL instances
wsl --unregister UbuntuRemove Ubuntu (deletes all data inside!)
wsl --updateUpdate WSL itself
wsl --set-version Ubuntu 2Upgrade distro to WSL version 2
07

Open Ubuntu Anytime

After setup, you can launch Ubuntu any of these ways:

  • Search "Ubuntu" in the Start menu
  • Open Windows Terminal and select Ubuntu from the tab dropdown
  • Type wsl in any PowerShell or Command Prompt window
  • Right-click a folder in File Explorer → "Open in Linux"
📂
Your Windows files are accessible inside WSL at /mnt/c/ (for C: drive). So cd /mnt/c/Users/YourName/Desktop takes you to your Windows Desktop.
section 05

First Steps After Install

Essential commands to run right after setting up Ubuntu — whether on a full install or WSL.

01

Update the system

bash
sudo apt update && sudo apt upgrade -y
02

Install essential tools

bash
sudo apt install -y curl wget git vim net-tools build-essential
03

Check your Ubuntu version

bash
lsb_release -a
